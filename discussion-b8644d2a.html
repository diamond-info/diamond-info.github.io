I suggest integration into the DMD wallet system for the exchange of messages and transfer data on the basis of open messenger qTox.

This can be done as a separate plug-in that allows you to exchange messages and data between two (in perspective, multiple) DMD network members.

As the address will be used with a DMD address of the owner entered in the address book. Exchange between random addresses not added to the address book is excluded.
