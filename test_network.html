DMD v4 test Network

Dmd v4 - start test network.

You need to have a web3 wallet that supports the DMD Diamond v4 Network, such as MetaMask Browser Extension or the Brave Browser.

* For Russian-speaking users I recommend using a portable EN version of Firefox and install MetaMask Extension EN (with English-language interface).

In order to avoid problems with connection settings, it is necessary to disable the proxy connection, or used Hide.me VPN service for this.

You need to add the DMD Network:

 CODE

Network Name:      DMDv4 Beta1
RPC URL:       https://beta-rpc.bit.diamonds
Chain ID:          27272
Symbol:            tDMD
Block Explorer URL:  https://beta-explorer.bit.diamonds

Now your wallet should be connected to the alpha4 testnet.

For receive test coins, it is necessary to convert your funds from the V3 wallet (see the detailed instructions here: Claiming-v4-Coins-User-Guide) on this page: Temp faucet link

Or by connect to our information channels #Discord or Telegram and we send you some tDMD for test staking at Test Staking Service.

All we request in return from alpha testers is that they give feedback and report issues.

More details about future DMD v4 can be found in whitepaper: Whitepaper
