2. Q:

What is a Diamond (DMD) wallet?

A:

A Diamond wallet is a program that runs in conjunction with all other Diamond wallets in the network and by consensus records and stores the contents of all wallets.

Other functions unique to Diamond are also regulated by the wallets, such as the creation and acceptance of MN and POS rewards.

This consensus record is known as the Diamond blockchain, and is similar to the consensus records (blockchains) maintained by other cryptocurrency networks.

The main advantage of DMD is that we use completely environmentally friendly technology which saves huge amount of electricity compared to POW coins.

13. Q:

How can I optimize my 1000 DMD to obtain maximum revenue from POS minting?

A:

Wallet has optional settings to determine the minimum group size coins when finding the block. For check your set size, you need to go into the console and run the command:

 CODE

If you see there is a value less than 100, then you need to change it. It is recommended to set this value to 200. For this you need to enter the command (unlock your wallet, before this action by entering the password):

 CODE

and restart the wallet:

After that your wallet will split the coins into groups not exceeding the value of 200 DMD.

15. Q:

What is the diamond file.conf and what is it for?

A:

The file diamond.conf is an important element of a wallet, allowing you to change the settings and configuration by adding in this file for more rows. There are quite a number of verbs supported by this file. Here I will give you the most basic:

 CODE

maxconnections=20

to limit the number of connections twenty

 CODE

onion=127.0.0.1:9150

The use TOR connection, via Advor or TOR browser with a specified port. Wallet works the simultaneously with protocol IPv4 and TOR. In many cases this increases the number of connections to the network.

 CODE

addnode=185.194.140.60

To add trusted nodes into the wallet. Used for rapid synchronization of the wallet with the network as a reliable base nodes for download the blockchain.

The addresses of these nodes are available on the website Block explorer

 CODE

port=9111

Forward port wallet inside the computer 17771 at free port (for example 9111). This may if you need to run several DMD wallets on the same computer.

 CODE

createwalletbackups=3

Limit on the auto backup for the file Wallet.dat. the default is 10 copies that are created each time you start wallet.

 CODE

staking=0
staking=1

To disable or enable POS minting. It is necessary for the actions associated with the wallet settings. Prevents you from accidentally sending coins to the steak

 CODE

changeaddress=dXdupf1YRAAhp2ENtMfF5CatHUVkPPC7tq

Use your own address for changeaddress. To prevent the generation of a new address in case of refund of the unused amount of the transaction.

The rest of the commands, you can get from the main wallet menu

17. Q:

19. Q:

I have a decent amount in DMD. what I need to do to protect my investments from sudden loss. Are there other ways of protection, in addition to creating a backup of the file wallet.dat?

A:

Yes. For create additional protection of your assets, it is recommended to keep copies of your private keys for each address that has the coins.

to do this, you need to fully unlock your wallet (by entering password), go to the console and execute the command:

 CODE

dumpprivkey You wallet address

If your wallet has many addresses with coins and you just want to get the lists of addresses with private keys, you need to enter the command:

 CODE

dumpwallet C:\test.txt

on drive C:\ will create the file test.txt listing all your addresses and private keys to them.

* Remember that all this information is very important and it should be stored in a safe place. The one who will be able to get the data of your private keys, you will be able to take control of all your coins.

23. Q:

What I need to do to run two wallet in one system?

A:

This is simple enough. All you need is to create two independent catalogue (for each the wallet must be used your file directory).

1. Close any instances of the wallet, press windowsbutton + R (for running), write:

 CODE

%appdata%

copy your existing directory to another directory for instance:

 CODE

d:\DMD2

2. Produce a shortcut on your desktop pointing to the already installed diamond-qt

3. Right click on the shortcut, choose properties, under Shortcut tab, in the Target: textbox, write the target directory for instance

 CODE

D:\Diamond-qt.exe -datadir=d:\DMD2

4. Go into diamond.conf in that (d:\DMD2) directory and add:

 CODE

port=17772

* The port can be any free port on your system.

The label needs to contain this line (startup with drive D:\).
