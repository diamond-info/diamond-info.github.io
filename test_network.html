Dmd v4 - start test network.

You need to have a web3 wallet that supports the DMD Diamond v4 Network, such as MetaMask Browser Extension or the Brave Browser.

You need to add the DMD Network:

 CODE

Network Name: DMDv4 Alpha4
RPC URL: http://rpc.uniq.diamonds:8540
Chain ID: 777012
Symbol: tDMD

Block Explorer URL:

Now your wallet should be connected to the alpha4 testnet.

Once you in that stage, just post your alpha4 testnet address on the #Slack or Telegram channel so we send u some tDMD which u then can use test staking at Test Staking.
